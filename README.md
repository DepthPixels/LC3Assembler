# LC-3 Assembler
A command-line assembler for LC-3 assembly language that converts .asm source files into binary machine code. Outputs .bin by default.

# Usage
## With py file or Executable
```bash
# Source Code
py main.py [options] <filename.asm>

# Compiled Executable
./LC3Assembler.exe [options] <filename.asm>
```
## Linux
```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/DepthPixels/LC3Assembler/refs/heads/main/run.sh) [options] <filename.asm>"
```
# Options
-h or --help for help.

-o or --obj for .obj output.
